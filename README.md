# trabajo-11-IAR-Liborio-Roman-Andres-Rubio
Implementación en Python de algoritmos de planificación de rutas: RRT, RRT*, Informed RRT* y BIT*. Incluye gestión de escenarios con obstáculos, detección de colisiones basada en radio de seguridad y experimentos detallados en Notebooks para replicación de resultados. Optimizado para la búsqueda de trayectorias óptimas en 2D.
